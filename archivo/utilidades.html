[Fortaleza Romhack]

¡La pides, la tienes!

Fortaleza Romhack es una web dedicada a recopilar información en castellano sobre cosas como documentación de la arquitectura interna de distintas consolas, mapas de memorias, lenguages ensamblador, y mucho más. Recopilamos también utilidades y otras cosas afines al romhacking. ¡Esperamos tu colaboración!

Utilidades

Estas son algunas de las utilidades más comunes utilizadas por los ROM Hacker y traductores:

Utilidades por HeXPLuS - Utilidades por Butz

Editores Hexadecimales

Translhextion en castellano - [686 KB]

Uno de los mejores editores hexadecimales, entre sus muchas funcionalidades están: cargar tablas thingy, extraer scripts, insertar scripts, permite hacer búsquedas relativas, etc. Una joya traducida a nuestro idioma por CiD.

Translhextion 1.6c - [543 KB]

La versión sin traducir de este maravilloso editor.

ThingyV - [91 KB]

Excelente modificación del editor hexadecimal Thingy realizada por Vegetal Gibber, entre las características más importantes está la posibilidad de abrir ISOS.

Editores de tiles

Tile Layer Pro - [244 KB]

Este excelente editor te servirá para editar las tiles de varios tipos de ROM, por ejemplo de NES, SNES, Genesis, Game Boy, etc. Recomendado.

Tile Molester - [448 KB]

Editor programado en Java, del mismo autor del anterior. Necesitas tener inslatada la Máquina Virtual de Java o JVM. Ésta la puedes descargar desde la página de Java.

Búsqueda relativa

SearchRX - [182 KB]

Este es uno de esos programas que te ayudará a la hora de empezar a hacer la tabla thingy. Con él puedes buscar en la ROM por palabras clave y así poder determinar el valor en hexadecimal de determinada letra o símbolo. Utilísima.

Tablas

Tabular - [69 KB]

Con esta herramienta puedes crear rápidamente la tabla thingy que vas a utilizar en tu traducción..

Aplicación y creación de parches

IPS Win - [246 KB]

Esta es una versión en windows de un programa que te permitirá aplicar los parches en formato IPS.

SNEStool - [14 KB]

Este programa te permitira realizar varias tareas con ROM de SNES, entre ellas crear parches en formato IPS, aplicar parches, unir archivos, etc. Imprescindible.

SEGAtool - [9 KB]

Al igual que el SNEStool esta es una herramienta para crear, aplicar parches, y otras funcionalidades impresindibles pero para los ROM de la consola SEGA.

Ensamblador, depuración y desarrollo.

Lunar Address - [47 KB]

Este programa básicamente es un convertidor de direcciones de SNES a PC y viceversa. Útil si tienes un log de alguna ROM y quieres localizar ésta dirección en el editor hexadecimal.

Lunar Expand - [40 KB]

Con esta pequeña utilidad podrás expandir el tamaño de la ROM y así lograr mayor espacio para colocar lo que desees.

SNES Professional ASM Development Kit - [436 KB]

Este programa permite la edición del código ensamblador en una ROM de SNES sin el uso de un programa ensamblador externo.

Hex2Snes - [394 KB]

Hex2snes es una utilidad que te permite realizar la conversión de direcciones de PC a direcciones de memoria en la SNES.

x816 - [43 KB]

X816 es un programa ensamblador que genera código máquina para la SNES.
