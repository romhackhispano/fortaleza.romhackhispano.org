Fortaleza Romhack es una web dedicada a recopilar información en castellano sobre cosas como documentación de la arquitectura interna de distintas consolas, mapas de memorias, lenguages ensamblador, y mucho más. Utilidades por Butz Klauser

Dark Devils Dragoon 1.0 - [82 KB]

Extractor magnífico de scripts que brinda las siguientes prestaciones:

  • Permite cargar archivos de gran tamaño tales como ISOS o Roms de N64
  • Permite visualizar directamente los textos de un juego con tan solo un clic
  • Es rápido (Full optimizaciones)
  • Hace dumpeos de texto con tan solo un clic
  • Maneja tablas thingy con valores de hasta dos bytes
  • El script extraído con el programa puede ser editado con cualquier editor de textos

En el archivo leeme.txt encontrarás una detallada explicación.

Slime Pointer 4.0.2 - [22 KB]

Con Slime Pointer podrás recalcular punteros para las siguientes consolas:

  • SNES
  • NES
  • GB
  • GBA
  • Punteros de Live a Live

Además de esto puedes sumar o restar cierta cantidad de bytes en el cálculo, para punteros de SNES puedes calcular punteros de 2 o 3 bytes, tiene un visualizador de punteros, etc. En realidad una herramienta utilísima.

Nutzor 1.1 - [10 KB]

Con Nutzor puedes extraer los scripts del afamado Dragon Warrior III, juego para la portátil Game Boy Color. Es una herramienta específica para esta ROM, ¡así que a disfrutarla!
