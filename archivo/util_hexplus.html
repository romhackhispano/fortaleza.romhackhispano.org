Xenoscript - Xenogears Script Editor

English: Xenoscript Editor is a script editor that allows translation of the SadNES cITy Xenogears Scripts easily. This utility has a preview window and several tools that allow the translation to other languages without complications.

Spanish: Editor de Scripts para el Xenogears, con este editor puedes traducir el Xenogears a portugués, francés, alemán, etc. Ésta es una versión beta, pero 100% funcional. Oficialmente el proyecto de traducción del Xenogears está acargo de Magimaster 2 y este servidor (HeXPLuS) en estos momentos.

Un agradecimiento muy especial a Magimaster 2, Sephiroth 1311 y a el gran Wiz por todas sus sugerencias .

Download / Descarga [Use 7zip to decompress]

  • Xenoscript 2.0.0.11 - [Lastest Binary File] - [79 KB]
  • Xenoscript Required Package - [1.32 MB]

Note: If you already have the Required Package you don't need to download it another time. The Required Package contains the following files (It doesn't contains the binary(.exe) file):
  • xenoscript.db (Scripts Database)
  • sqlite3.dll (Required dinamic library)
  • Font folder (Font.bmp file in)
  • run_me.bat (To register the sqlite3 dll)
  • Xenoscript User Guide.pdf (idem)

Hextractor - Extractor de scripts

Hextractor v.1.0.0.6 - [162 KB]

Este pequeño programa te servirá para extraer scripts de una manera muy cómoda. Está hecho en una ventana de manera que puedas utilizarlo en conjunto con el Translhextion, para que vayas anotando cuales son los rangos de offsets de scripts que deseas extraer. A partir de la versión 1.0.0.6 puedes indicar el valor de un byte que indica el inicio de un bloque de texto, si la ROM de la que deseas extraer scripts tiene un byte que indique el inicio de un bloque de texto, pues este es el byte que tienes que colocar en esa caja de texto. Recomiendo leer el archivo Leeme.txt.

Un agradecimiento muy especial a Dark-N por todas sus sugerencias y reportes de error.

Vrecalc IGU - Front End para el Vrecalc

Vrecalc IGU v.1.0 - [23 KB]

Interfaz Gráfica de Usuario para el Vrecalc de Vegetal Gibber. Con este programa podrás utilizar este recalculador de punteros con una interfaz en 32 bits de manera muy cómoda, solo basta copiar el Vrecalc.exe en el mismo directorio de este programa.

Minieditor - Minieditor de Scripts muy básico

Mini-editor v.1.0.0.5 - [16 KB]

Éste es un editor de scripts que programé para traducir los scripts del Soul Blazer de una manera cómoda, ya que con mi programa Hextractor (utilidad también programada para el Soul Blazer), los scripts extraídos son aproximadamente 400. Este editor permite una movilidad genial entre script y script. Creo que puede ser utilizado para editar otros scripts que no hayan sido extraídos con el Hextractor, y creo que Lukas lo ha utilizado en parte de uno de sus proyectos.

Entre las ventajas que tiene están:

  • Excelente movilidad entre script y script.
  • Opción de guardado rápido, con tan solo presionar la tecla F12, tu script será guardado.
  • Opción de eliminar scripts inservibles rápidamente con solo presionar la tecla SUPR.
  • Seguimiento de líneas, control de espacio total, por línea y advertencias sugeridas.
  • [Nuevo] Puedes insertar un marcador y regresar luego al archivo por donde habías quedado.

Etiquetanator - Modificador de tags masivo

Etiquetanator v.1.0.0.0 - [10 KB]

Etiquetanator es un programa que hice para cambiar de un tipo a otro las etiquetas o llaves de cierre utilizadas en los scripts. Este programa lo hice pensando en la necesidad de migrar a otro tipo de etiquetado para utilizar programas de inserción de scripts que utilizan ese tipo de etiquetado.

Puedes cambiar entre diferentes formatos, mira los detalles en el Leeme.txt que viene en el fichero.

Importante:

  • Permite cambiar en segundos las etiquetas de TODOS tus scripts por las deseadas, sin tener que estar abriendo archivo por archivo.

MiMTE - Optimizador MTE

MiMTE v.1.0.0.0 - [15 KB]

Este diminuto programa te servirá para analizar tus scripts totalmente traducidos y ver qué palabras completas son las que más se repiten en todo el libreto, y así generar tu propio diccionario MTE. El programa genera dos archivos: una tabla Thingy con los resultados, y un archivo llamado Frencuencias.txt que detalla la frecuencia de repetición de cada palabra del libreto. Lee más detalles en el archivo Leeme.txt incluido en el ZIP.

Importante:

  • Permite analizar varios archivos como si de sólo uno se tratase. :)
